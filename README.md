# LED driver PT4115 
Projekt sterownika LED przeznaczonego do sterowania lampą LED.
Dzięki możliwości regulacji mocy światła za pomocą sygnału PWM układ ma zastosowanie w doświetlaniu roślin na rozsadniku.
